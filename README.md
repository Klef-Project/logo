# Klef-Project/logo

## 概要

本リポジトリは, Klef Projectおよびその傘下のプロジェクトに関連するロゴを集約するものである. 
これらのロゴは CC BY-ND ライセンス下において頒布されるものとする. 
また, 使用の際にはライセンスガイドラインを参照いただきたい. 

## ファイル

本リポジトリにはプロジェクトごとにディレクトリが用意されている. 
現在, 以下のファイルが配置されている. 

- klef
  - klef-color-logo-only.png : Klefロゴ (カラー, PNG形式)
  - klef-color-logo-only.svg : Klefロゴ (カラー, SVG形式)
  - klef-logo-inkscape.svg : Klefロゴ (Inkscape SVG形式)
- klave
  - klave-color-logo-only.png : Klaveロゴ (カラー, PNG形式)
  - klave-color-logo-only.svg : Klaveロゴ (カラー, SVG形式)
  - klave-logo-inkscape.svg : Klaveロゴ (Inkscape SVG形式)
  - klave-wb-logo-only.png: Klaveロゴ (モノクロ, PNG形式)
  - klave-wb-logo-only.svg: Klaveロゴ (モノクロ, SVG形式)
- README.md : このファイル
- LICENSE : ライセンスファイル
- guideline.md : ライセンスガイドライン
