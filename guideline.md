# Klave Project ライセンスガイドライン

## 第2章 ロゴマーク

本プロジェクトに係るロゴマークは, 原則としてクリエイティブ・コモンズ 表示ー改変禁止 4.0 ライセンス (CC BY-NDライセンス) のもとに公開されるものとします. 

このため, 改変をしない限りは, 営利・非営利をとわず, 自由に使用することができます. 

### 改変について

また, CC BY-NDライセンスでは認められていない改変についても, 本プロジェクトにおいては, 運営がその必要性を認められる場合に限り, 以下の手順を踏むことにより使用することができる場合があります. 

1. Inkscape SVG (\*-inkscape.svg) 形式のファイルをダウンロード・編集する
2. PNGおよびPlain SVGに書き出す
3. 書き出したファイルを添付したIssueを作成する. 
4. 運営側がリポジトリにファイルを追加し, IssueがCloseされる

この手順により承認されたファイルは新たな著作物としてCC BY-NDライセンスが適用されるものとし, 本リポジトリにおいて公開されるものとします. 改変が認められるのは, 運営が提供するInkscape SVG形式のファイルをもとにする場合のみであることに注意してください. 
なお, 当該ファイルは`variants`ディレクトリに配置されます. 使用する際は, 必ず`variants`ディレクトリからダウンロードするようにしてください. 

`variants`ディレクトリに追加される前に使用した場合はライセンス違反となります. 

改変が認められる部分は以下であり, それ以外の改変は認められないので注意してください. 

- 配色
- Strokeの太さ

なお, 上記手順を踏んだ結果, 本プロジェクトの運営 (=本リポジトリのCODEOWNERS) がそのファイルの使用を認めない (=その変更を棄却する) 場合があります. その場合, 当該Issueは「Close as Not Planned」でCloseされますので, 使用前に必ず確認するようにしてください. 

> [!IMPORTANT]
> あくまでこの条項は, CC BY-NDライセンスに基くものではなく, 本プロジェクトの独自の追加要項である点に注意してください.
